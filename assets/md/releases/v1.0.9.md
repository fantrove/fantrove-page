### New

- **Static build pipeline for HTML pages**
  Pages are now pre-built at deploy time rather than relying entirely on client-side JavaScript rendering. This improves initial load performance and allows search engines to crawl fully-rendered content.

### Improved

- **Meta tags and SEO structure**
  Added proper meta descriptions, Open Graph tags, and structured heading hierarchy across all pages. Each page now has unique, descriptive metadata for better search engine indexing.

- **Page load performance**
  Optimized script loading order and added preloading for critical CSS resources. Pages now render meaningful content faster, reducing the time to first meaningful paint.
