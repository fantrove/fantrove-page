**TL;DR** — ระบบบล็อก 4 ชั้นทำงานได้จริงแค่ครึ่งเดียว: การตรวจสอบใน CI ที่ควรบล็อกตอนลืม bump เวอร์ชั่น อ่านไฟล์ commit เดียวกันมาเทียบกับตัวเองเสมอ จึงแยกไม่ออกระหว่าง "bump จริง" กับ "ไม่ได้ bump" ส่วน git hooks ฝั่ง local ตอนนี้ติดตั้งอัตโนมัติแล้ว แทนที่จะพึ่งขั้นตอนที่ต้องรันเองแล้วมักลืม

## เกี่ยวกับการแก้ไขนี้

ระบบนี้ควรบล็อก commit, push หรือ deploy ถ้าไม่มีการเปลี่ยนหมายเลขเวอร์ชั่น แต่เมื่อตรวจสอบประวัติโปรเจกต์จริงพบว่าชั้นนี้ไม่เคยทำงานสำเร็จเลยสักครั้ง — ไม่มี commit จาก CI แม้แต่ commit เดียวในประวัติกว่าพันครั้ง สาเหตุคือบั๊กคลาสเดียวกับที่แก้ไปใน 2.2.3: การตรวจสอบใน CI เทียบเวอร์ชั่นใน current.md กับ current.md ที่อ่านจาก git HEAD แต่ใน CI นั้น HEAD คือ commit ที่มีการ bump ไปแล้ว ทำให้ทั้งสองฝั่งเหมือนกันเสมอ ระบบจึงแยกไม่ออกว่า "เวอร์ชั่นเปลี่ยนจริง" กับ "ลืม bump" — ผลคือจะบล็อกทุกครั้งอย่างเงียบๆ โดยไม่มีใครรู้ตัว

นอกจากนี้ pre-commit และ pre-push hook ฝั่ง local ที่ควรจับการลืม bump ก่อนแม้แต่จะ push ก็ทำงานได้ก็ต่อเมื่อนักพัฒนารันคำสั่งติดตั้งด้วยตัวเองครั้งหนึ่งก่อน ซึ่งลืมง่ายมาก โดยเฉพาะตอน clone โปรเจกต์ใหม่

### Fixed

- **การตรวจสอบ version bump ใน CI เทียบกับเวอร์ชั่นก่อนหน้าจริงๆ แล้ว**
  ระบบตอนนี้ตรวจก่อนว่า git HEAD ตรงกับไฟล์บน disk หรือไม่ ถ้าตรง (กรณี CI หลัง push) จะย้อนดูประวัติ git ของไฟล์นั้นเพื่อหา commit ที่มีเวอร์ชั่นก่อนหน้าจริงๆ แทนที่จะเทียบ commit กับตัวเอง ทดสอบแล้วทั้งกรณี bump จริง (ผ่าน) และกรณีลืม bump (บล็อกถูกต้อง)

- **Git hooks ติดตั้งอัตโนมัติแล้ว**
  การตรวจสอบ pre-commit และ pre-push ตอนนี้ติดตั้งตัวเองอัตโนมัติตั้งแต่ครั้งแรกที่ติดตั้ง dependencies เหมือนกับที่โปรเจกต์ Node.js ส่วนใหญ่ทำ แทนที่จะพึ่งคำสั่งแยกที่ต้องรันเอง การติดตั้งจะข้ามตัวเองอย่างปลอดภัยเมื่ออยู่ใน CI หรือนอก git repository เพื่อไม่ให้กระทบ automated build
  ระบบยังคงสร้างไฟล์ประวัติในโฟลเดอร์ releases อัตโนมัติเมื่อมีเวอร์ชั่นใหม่ นักพัฒนาไม่ต้องสร้างหรือจัดการไฟล์ในโฟลเดอร์ releases เอง แค่แก้ไข current.md เท่านั้น

### สิ่งที่คุณจะสัมผัสได้

- โครงสร้างไฟล์ชัดเจน — current.md แยกจาก history
- นักพัฒนาทำงานง่ายขึ้น — แก้ไขแค่ current.md เท่านั้น
- ประวัติยังคงถูกบันทึกอัตโนมัติเหมือนเดิม
