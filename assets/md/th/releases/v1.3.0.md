### New

- **PopupSystem v2.0 — universal popup shell**
  โครงสร้าง popup ครบถ้วนพร้อม shadow DOM isolation, 8 presets (alert, confirm, toast ฯลฯ), event bridge และรองรับ theme ทุก popup UI ใช้ระบบนี้แทนการเขียน inline

- **version-core.js — popup แจ้งเตือนอัพเดท**
  ผู้ใช้จะเห็น popup แจ้งเตือนสไตล์สวยงามเมื่อมีเวอร์ชั่นใหม่ แสดงเลขเวอร์ชั่น สรุปอัพเดท สูงสุด 4 รายการ และลิงก์ไปหน้า What's New รองรับการปิดและตรวจจับ idle session

- **whats-new.json — ข้อมูล release แบบมีโครงสร้าง**
  ไฟล์ JSON สำหรับ release ปัจจุบัน รองรับ i18n (en/th), sections แบบ typed (new/improved/fixed) และคำอธิบายรายละเอียด

- **version.json — ติดตามเวอร์ชั่นทั่วทั้งแอป**
  ไฟล์เวอร์ชั่นง่ายๆ ที่หน้า What's New poll ทุก 60 วินาทีเพื่อตรวจจับการอัพเดทใหม่

### Improved

- **popup.js โหลดบนทุกหน้า HTML ทั้ง 10 หน้า**
  ทุกหน้าของเว็บไซต์โหลด popup.js ก่อน version-core.js แล้ว ทำให้ระบบแจ้งเตือนอัพเดททำงานถูกต้องบนทุกหน้า

- **หน้า What's New พร้อม relative timestamps แบบ real-time**
  หน้า What's New แสดง "เมื่อกี้", "5 นาทีที่แล้ว" ฯลฯ พร้อม ticking แบบ real-time ที่เปลี่ยนเป็นวันที่เต็มหลัง 10 วัน รองรับทั้งอังกฤษและไทย

### Fixed

- **Popup แจ้งเตือนอัพเดทไม่แสดงบนหน้าใดเลย**
  popup.js ไม่ถูก include ในหน้า HTML ใดเลย ทำให้ version-core.js ข้ามการแสดง popup หลังรอ 5 วินาทีเงียบๆ ได้เพิ่ม script tag ให้ทุก 10 หน้าแล้ว
