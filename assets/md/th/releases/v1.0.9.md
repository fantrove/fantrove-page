### New

- **Static build pipeline สำหรับหน้า HTML**
  หน้าเว็บถูกสร้างล่วงหน้าตอน deploy แทนที่จะพึ่งพา JavaScript rendering ที่ฝั่ง client ทั้งหมด ช่วยเพิ่มประสิทธิภาพโหลดหน้าครั้งแรก และให้ search engine สามารถ crawl เนื้อหาที่ render แล้วได้

### Improved

- **Meta tags และโครงสร้าง SEO**
  เพิ่ม meta description, Open Graph tags และลำดับ heading ที่เหมาะสมให้ทุกหน้า แต่ละหน้ามี metadata เฉพาะของตัวเองสำหรับการ index ที่ดีขึ้น

- **ประสิทธิภาพการโหลดหน้า**
  ปรับลำดับการโหลด script และเพิ่ม preloading สำหรับ CSS ที่จำเป็น หน้าแสดงเนื้อหาที่มีความหมายเร็วขึ้น ลดเวลา first meaningful paint
