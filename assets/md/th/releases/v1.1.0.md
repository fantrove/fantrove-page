### New

- **รองรับหลายภาษา (อังกฤษ / ไทย)**
  เว็บไซต์รองรับสองภาษาพร้อมตัวสลับภาษา ผู้ใช้สามารถสลับระหว่างอังกฤษและไทยได้ และการตั้งค่าจะถูกบันทึกไว้ ข้อความ UI ป้ายนำทาง และเนื้อหาจะปรับเปลี่ยนอัตโนมัติ

- **ระบบ language proxy และ sync**
  Language proxy (lang-proxy.js) และระบบ sync (lang-sync.js) ช่วยให้สถานะภาษาสอดคล้องกันทุกหน้าและทุก script การเปลี่ยนภาษาจะ trigger custom event ที่ component ทั้งหมดรับฟัง

### Improved

- **Navigation bar รองรับหน้าจอเล็ก**
  แถบนำทางด้านบนปรับตัวบนหน้าจอเล็กได้ดีขึ้น พร้อมการปรับขนาด layout และปุ่มที่ใช้สัมผัสได้ง่ายขึ้น

- **ความราบรื่นของการเปลี่ยนหน้า**
  การเปลี่ยนหน้าถูกปรับปรุงให้ลด visual jank และให้ประสบการณ์ที่ราบรื่นขึ้นเมื่อนำทางระหว่างส่วนต่างๆ
