FantroveVerse Loader (FVL)

Framework หน้าโหลดแยกออกจาก nav-core — รองรับ 4 display modes, lightweight, zero dependencies, ทำงานลื่นไหลบนทุกสเปค

① Fullscreen Mode

Overlay เต็มหน้าจอ — ใช้ตอนเปลี่ยนหน้าหรือโหลดข้อมูลใหญ่ เหมาะกับ fullscreen transitions

// ค่า default — ใช้ singleton ID
const h = FVL.show({
  message: 'กำลังโหลด...',
  autoHideAfterMs: 4000
});
h.hide();

② Scoped Mode

ครอบ container เฉพาะ เช่น การ์ด/section ใดๆ ไม่เต็มจอ แค่ครอบพื้นที่ที่กำหนด

FVL.scoped({
  target: '#demo-scoped-box',
  message: 'Fetching...'
});

③ Inline Mode

เล็ก แทรกในปุ่ม/element ใดๆ — เหมาะกับ submit button หรือ action button

FVL.inline({
  target: '#my-btn'
});
// หรือ replace content:
FVL.inline({
  target: '#my-btn',
  replaceContent: true,
  message: 'Working...'
});

④ Topbar Mode

Progress bar บางๆ ติดด้านบน viewport — เหมือน NProgress เหมาะกับ navigation ระหว่างหน้า

// indeterminate:
FVL.topbar();

// determinate (0..1):
const h = FVL.topbar({ progress: 0.3 });
h.setProgress(0.7);
h.hide();

Backward Compatibility

API เดิมจาก nav-core ทำงานเหมือนเดิมผ่าน proxy:

// เหล่านี้ทำงานเหมือนเดิม:
window.showInstantLoadingOverlay('Loading...');
window.removeInstantLoadingOverlay();
window._navCore_contentLoadingManager.show();
NavCoreModules.LoadingService.hide();
